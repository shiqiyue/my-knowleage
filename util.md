## ghproxy
```
国内访问github资源时候，有时候会访问不上，可以通过ghproxy代理访问
比如 git clone https://github.com/vegasbrianc/prometheus.git 
换成 git clone https://ghproxy.com/https://github.com/vegasbrianc/prometheus.git
```
